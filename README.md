# To-Do-List-CLI-
Command-line To-Do List that lets users add, view, and delete tasks during a session.
Problem Statement:  
Create a command-line To-Do List app that lets users add, view, and delete tasks during a session.
- List operations in Python  
- Using loops and conditionals  
- Handling user input  
- Basic CLI app logic

#What is this program doing overall?
1. Store tasks
The program keeps all tasks in a list during the session.

2. Show menu
It displays options like:
a) Add task
b) View tasks
c) Mark the task as complete
d) Delete task
e) Exit

3. Take user input
The user enters a number to choose what they want to do.

4. Decide action
The program checks the user’s choice and decides which action to perform.

5. Perform action
Depending on the choice:
Add → saves a new task
View → shows all tasks
Complete → marks a task as done
Delete → removes a task

6. Repeat
After completing the action, the program shows the menu again.

7. Exit
When the user selects Exit, the program stops.

#Simple Summary👉  “Show menu → take input → perform action → repeat until exit.”
